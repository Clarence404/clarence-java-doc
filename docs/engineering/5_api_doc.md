# API 文档

> Spring Boot 中的 API 文档方案，详细实践见 [spring-boot/9_api_doc](../spring-boot/9_api_doc)。

## 方案对比

| 方案 | 优点 | 缺点 |
|------|------|------|
| Springdoc OpenAPI 3 | 标准化、活跃维护 | 注解略多 |
| Knife4j | 国内友好、UI 美观 | 依赖 Swagger2 |
| 手写 API 文档 | 灵活 | 维护成本高 |
