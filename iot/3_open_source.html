+---------------------------------------------------+
|               数据分析与应用层                    |
|  - 告警规则引擎                                   |

|  - 流处理引擎（Flink）：实时计算告警              |

+---------------------------------------------------+
|               数据传输层                          |
|  - 消息队列（Kafka）：高并发数据缓冲              |
+---------------------------------------------------+
|               数据采集层                          |
|  - 设备与传感器：数据生成                         |

2、微服务架构

+--------------------------------------------------------------+
|                        设备端 (Device)                        |
|   - 智能水表、摄像头、网关设备（通过 MQTT/HTTP/CoAP）             |
+--------------------------------------------------------------+
                                ||
                                ||  MQTT/HTTP/CoAP
                                \/
+---------------------------------------------------------------+
|                  设备接入层 (Access Layer)                    |

|   - 设备消息解析 (边缘服务，Micronaut 小服务)                    |

+---------------------------------------------------------------+
                                ||
                                ||  设备数据流、控制指令
                                \/
+---------------------------------------------------------------+
|                 边缘计算层 (Edge Layer)                        |

|   - 规则引擎、初步数据清洗                                      |
|   - 事件触发（如告警推送）                                      |
|   * 就近处理，快速响应                                          |
+---------------------------------------------------------------+
                                ||
                                ||  业务数据流、异步消息 (Kafka/RabbitMQ)
                                \/
+---------------------------------------------------------------+

|   - 流量治理（Istio，灰度发布，限流熔断）                      |
|   - 配置中心（Nacos）                                           |
|   - 消息队列中心（Kafka，EMQX MQTT Broker）                    |
|   - 数据持久化（TSDB，关系型数据库）                           |
|   * 大数据处理，业务系统集成                                    |
+---------------------------------------------------------------+
                                ||
                                ||  报表系统、运维平台、开放 API

+---------------------------------------------------------------+
|                   运维管理 & 第三方接入                         |
